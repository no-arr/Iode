**Iode** is an unofficial port of the Godot game engine to iPad, built entirely in C# using .NET 8. It runs on top of the Godot game editor and is currently in the very early stages of development. Iode is both an ambitious technical experiment and a personal learning project.

---

## 🚀 Why?

Apple’s M-series chips are insanely powerful and efficient—making iPads compelling devices for real creative work. But the biggest gap compared to Macs is the operating system: iPads are locked into iPadOS, while Macs use macOS.

As a first-year college project, I thought—why not try something bold?

> Can I turn an iPad into a full game development environment?

That’s where Iode comes in.

---

## 📥 Installation

You can sideload Iode to your iPad without jailbreaking, using any of these tools:

- [SideStore](https://sidestore.io/)
- [Sideloadly](https://sideloadly.io/)

Just download the latest `.ipa` file (not yet released), and install it using your preferred sideloading method.
